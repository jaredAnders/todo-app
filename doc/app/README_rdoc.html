    Zenshin¶ ↑

    Zenshin is a Todo SPA built using Rails 4.1.9 and jQuery 1.11.1 to fulfill part of the Firehose Project curriculum. The primary focus of this project was learning how to set up a Rails API controller and consume it via Ajax requests.
